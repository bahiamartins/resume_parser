<h2>Resume Parser</h2>
A python script that extracts name, email and phone number from resumes using spacy model.

<h2>Pacakges to install:</h2>
Spacy
<br>
Pandas
<br>
Numpy
<br>
Xlsxwriter
<br>
pdfminer
<br>
bs4
<br>
urllib2

<h2>After installing spacy package, we need to download:</h2>
python -m spacy download xx
